# PortfolioX Rebuild

Paquete reconstruido generado a partir de los archivos disponibles en el workspace y la documentación funcional existente.

## Incluye
- Estructura Vite + React
- AuthContext, useAuth y ProtectedRoute
- usePortfolio, useTransactions y usePriceUpdater
- FundsTable y TransactionHistory
- Cliente Supabase
- Types base
- Pages mínimas reconstruidas
- Migraciones SQL iniciales

## Importante
Algunos componentes faltaban en el workspace original, así que esta entrega incluye placeholders funcionales mínimos para completar el paquete descargable.
